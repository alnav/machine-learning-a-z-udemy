# Multiple linear regression

    y = b0 + b1x1 + b2x2 + b3x3 + ... + bnxn

- b0 = constant
- bn = coefficients
- xn = independent variables

## Assumptions for multiple linear regression
1. linearity
2. homoscedasticity
3. multivariate normality
4. independence of errors
5. lack of multicollinearity
